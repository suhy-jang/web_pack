## Deployment

- [Online Deployment](https://shjang7.github.io/old_school/)

## Getting started

1. open browser with server run
```
$ npm run build
$ npm run server
```

2. In browser, check console log
- ctrl + c
- 'console' click
- check 3 lines of message
